## 1. Цель

Получить приближенную площадь пересечения трёх окружностей методом Монте-Карло, сравнить точность двух способов (широкая и узкая область выборки), построить графики зависимости оценки площади и относительной ошибки от количества точек.

---

## 2. Постановка задачи

**Окружности:**

1. Центр (1, 1), радиус 1  
2. Центр (1.5, 2), радиус sqrt(5)/2  
3. Центр (2, 1.5), радиус sqrt(5)/2  

**Точная площадь:**

S_exact = 0.25 * pi + 1.25 * asin(0.8) - 1  
Приблизительное значение: S_exact ≈ 0.944517

**Оценка методом Монте-Карло:**

- Пусть M — число точек, попавших в пересечение всех трёх окружностей.  
- Пусть N — общее количество случайных точек.  
- Пусть |R| — площадь выбранного прямоугольника.  

Тогда оценка площади:  
S_est = (M / N) * |R|

**Используемые области:**

- **Широкая:** прямоугольник [0, 3] × [0, 3]  
- **Узкая:** минимальный bounding box всех трёх окружностей

## 3. Реализация

- Использован диапазон N = 100, 600, 1100, ..., 100000  
- Для каждого N вычислены площадь и относительное отклонение  
- Данные записаны в файл `data.csv`  
- Построение графиков выполнялось через `plt.py`

- Ошибка убывает примерно как 1 / sqrt(N)
