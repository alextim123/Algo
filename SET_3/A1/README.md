## 1. Цель
Приблизить площадь пересечения трёх окружностей методом Монте-Карло, сравнить точность при использовании широкой и узкой областей выборки, построить графики зависимости оценки и относительной ошибки от числа точек.

---

## 2. Постановка задачи
Окружности:
- \(C_1=(1,1), r=1\)
- \(C_2=(1.5,2), r=\sqrt{5}/2\)
- \(C_3=(2,1.5), r=\sqrt{5}/2\)

Точная площадь:
\[
S_{\text{exact}} = 0.25\pi + 1.25\arcsin(0.8) - 1 \approx 0.944517.
\]

Оценка методом Монте-Карло:
\[
\tilde S = \frac{M}{N} \cdot |R|.
\]

Области:
- **Широкая:** \([0,3]\times[0,3]\)
- **Узкая:** минимальный bounding box всех окружностей.

---

## 3. Реализация
- Сгенерировано \(N = 100, 600, \dots, 100000\).
- Для каждого \(N\) вычислены площадь и относительная ошибка.
- Данные сохранены в `data.csv`, обработаны через обновлённый `plt.py`.
- Графики сохранены как Figure_1 и Figure_2.

---

## 4. Результаты

### Приближённая площадь (Figure 1)
- Оценки колеблются вокруг точного значения.
- При малом \(N\) разброс значительный.
- Узкая область даёт более стабильные и точные значения.

### Относительное отклонение (Figure 2)
- Ошибка убывает примерно как \(1/\sqrt{N}\).
- Узкая область почти всегда точнее широкой.
- При \(N > 40000\) ошибка обеих методов падает к ~1%.
